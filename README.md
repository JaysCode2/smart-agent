# smart-agent

这是一个由脚手架生成的 Spring Boot 初始项目，包含简单的示例控制器与配置，便于快速开始开发。

快速开始：

1. 构建并运行（需要本地安装 Maven & JDK 17+）：

```bash
mvn clean package
mvn spring-boot:run
```

2. 访问示例接口：

打开浏览器访问 http://localhost:8080/ 会看到文本响应："Smart Agent is running"

3. 下一步建议：
- 根据业务调整 `groupId`/`artifactId` 和 `package`。
- 添加更多模块、数据库配置和安全配置。

大模型应用开发，涉及agent、rag、mcp、function call、reAct、plan-execute-replan等
